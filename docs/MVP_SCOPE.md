# BuilderMint MVP Scope

## In scope

- Landing page
- Supabase-ready auth with dev fallback
- Skill profile CRUD
- Opportunity generation (5 ideas)
- Opportunity detail with scoring
- MVP build plan generation
- Launch plan generation
- Dashboard navigation
- Mock AI provider for local development
- xAI provider when API key is configured

## Out of scope

- Stripe billing
- Team workspaces
- Marketplace
- Complex web scraping
- Public API
- Codra Deploy integration (planned later)
